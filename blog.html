What is DOM and what is the purpose of DOM?


DOM means Document Object Model. It is a sort of model which can be manipulated by using JavaScript. This model is a depiction of the entire HTML but in a form of object or nodes which enable the programming language for better interactivity. By using this object based concept of the webpage we can make the website more interactive.

How will you select HTML elements using DOM? Name the DOM methods.


There are few ways we can select HTML elements: by ID name, by TAG name, by CLASS name. Some of the methods are as follows:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

What is a event bubble?


Event Bubbling is a concept similar to when someone exhales underwater the bubble moves upwards. In the DOM, similar thing happens when a event is triggered inside a child element or object the event gradually move upwards to the parents of the child node, hence the similar event is triggered in the parent as well.

What is a callback function and why will you use it?


Callback functions are those which can be used as a parameter to another functions. Because the functions act as a object in JavaScript we can pass the function as parameter, the same way we pass normal objects.
